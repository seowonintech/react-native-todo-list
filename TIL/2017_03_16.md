## 배운 점

## 이슈

* TextInput value 컨트롤을 하려면 redux state구조를 사용해야 될 듯

```javascript
<TextInput
    style={{height: 40, borderColor: 'gray', borderWidth: 1}}
    onChangeText={text => value = text}
    ref={node => {
        textInput = node
    }}
    value={value}
/>
<Button
    title="Add"
    onPress={ () => {
        console.warn('[TS_Warning] value : ' + value);
        dispatch(addTodo(value));
        this.value = ' ';
    }}
/>
```
